The `Attachments/` folder contains all non-Markdown files in this vault: images, [[Excalidraw]] drawings, PDFs, and any other binary assets.

## Conventions

- All attachments go directly in `Attachments/` — no sub-folders.
- File names should be descriptive. For Excalidraw files, use the name of the concept being diagrammed (e.g. `auth-flow.excalidraw`).
- To embed an attachment in a note use `![[filename.ext]]`.

## Configured automatically

[[Obsidian shared settings]] configures Obsidian to save new attachments and new Excalidraw drawings to this folder automatically. Contributors do not need to move files manually.
