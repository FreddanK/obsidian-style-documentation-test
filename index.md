# Documentation

Welcome to the internal documentation vault. This site is generated from a [[Git]]-versioned [[Obsidian]] vault and published with [[Quartz]].

## How to navigate

- Use the **search bar** (top right) to find notes by keyword.
- Use the **graph view** to explore how notes connect.
- Browse by **tag** using the tag index pages.
- Use the [[Maps Of Content]] as curated starting points.

## How to contribute

See [[CONTRIBUTING]] for step-by-step instructions. Short version:
- **Read**: just use this site.
- **Quick edit**: use the GitHub web editor.
- **Regular writing**: set up [[Obsidian]] locally with the shared settings.
